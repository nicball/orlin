# orlin
A statically-typed compiled LISP dialect just for fun.

### Supported Features

* Int/unit data types.
* Type inference.
* Local variables.
* Lambda expressions and closures.

### Features Underway

* Garbage collection.
* Better recursion support.
* Type synonyms.
* Variants (including syntax-sugared enums).
* Records (including syntax-sugared tuples).
* If branch.
